# Day 03 - Linux Basics

## Commands I Learned Today

### | Command | What it does |

| grep | Search for a pattern inside a file |
| chmod | Change file permissions |
| ps | Show running processes |
| df | Show disk space usage |
| top | Live view of system resource usage |

#### example:
grep "error" logfile.txt        # find "error" in a file
chmod 755 script.sh             # make a script executable
ps aux                          # list all running processes
df -h                           # disk usage in human-readable format
top                             # live CPU/memory monitor (press q to quit)
